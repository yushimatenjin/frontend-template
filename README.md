

## ローカルでの開発

ローカルで開発を行う場合は、以下の手順に従ってください：

1. リポジトリをクローンします：
   ```
   git clone https://github.com/yourusername/◯◯◯.git
   cd ◯◯◯
   ```

2. 依存関係をインストールします：
   ```
   npm install
   ```

3. 開発サーバーを起動します：
   ```
   npm run dev
   ```

4. ブラウザで `http://localhost:3000` を開きます。
